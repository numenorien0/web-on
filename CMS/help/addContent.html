Création des pages

Vous vous trouvez dans la création d'une page de votre site. Nous allons voir ensemble comment nous pouvons mettre en ligne et dynamiser une de vos pages.

Vous constaterez que Geronimo est nativement multilingue. En cliquant sur les différents onglets, vous avez le choix d'encoder les autres langues.

Vous devez encoder le nom de votre page ici. C'est le titre principale de votre page. Il apparaitra sur la majorité des "miniatures" ainsi que sur presque toutes les pages. C'est aussi ce nom la qui sera affiché dans votre barre de navigateur.
