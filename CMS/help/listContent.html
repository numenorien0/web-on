Structure des pages

Nous allons voir ensemble comment un site fait avec Geronimo est structuré.

Ici se trouve la liste de vos pages. Pour simplifier la compréhension nous avons hiérarchisé les pages. Nous considérons qu'une page article est dépendante d'une page "Actualité" par exemple. La pages peuvent donc s'imbriquer les unes dans les autres.

En cliquant sur le nom d'une page, vous parcourez ainsi son arborescence. D'ailleurs, vous constaterez que les pages ayant une arborescence ont une icone en forme de dossier.

Le principe est assez simple. Les pages imbriquées sont des sous-pages. On pourrait les assimiler aux différents blocs qui composent la page. Leur mise en page se modifie directement dans l'édition. Ainsi vous pouvez choisir d'afficher ce bloc d'une certaine façon et un autre différemment. Ce qui est génial, c'est qu'il n'y a pas de limite dans l'imbrication. Ainsi, vous pouvez avoir une page "actualité" dans laquelle on retrouve les sous-pages "coups de coeur" et "sports" (qui ont tous les deux une mise en page propre) dans lequels on peut ajouter des articles. Les possibilités sont infinies avec Geronimo.

Alors que les pages qui n'ont pas de sous-page ont une icone de "fichier"

En cliquant sur modifier, vous modifiez la page sélectionnée.

Un clic droit sur les pages vous donne des possibilités de suppression, modification, déplacement, etc.
Les pages sont affichées dans l'ordre ou elles apparaissent dans cette page. Pour changer cet ordre, glissez et déposez. C'est aussi simple que ça.
